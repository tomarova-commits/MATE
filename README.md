# MATE Talent Card

Простейший статический сайт для GitHub Pages: карточка таланта, подрядчика, студии или компании.

## Файлы

- `index.html` — структура страницы и поля карточки.
- `styles.css` — адаптивные стили без фреймворков.
- `script.js` — базовый JS: переключение статуса, копирование контакта и светлая/темная тема.

## Дизайн-система

Страница использует CSS-токены из дизайн-системы: `background`, `foreground`, `card`, `primary`, `secondary`, `muted`, `accent`, `destructive`, `border`, `ring`, радиус `0.5rem` и базовые тени.

Шрифты подключены без Next.js и сборки через Google Fonts:

- Montserrat — основной текст;
- Merriweather — заголовок;
- Source Code Pro — технические подписи и управляющие элементы.

## Как отредактировать карточку

Откройте `index.html` и замените примерные данные на реальные:

- имя, фамилию или название студии/компании;
- род занятий;
- контакт;
- описание;
- статус `Работали / Не работали`;
- ставку, гонорар или сумму проекта;
- комментарии.

## Как опубликовать на GitHub Pages

1. Внесите изменения в ветку `main`.
2. Сделайте коммит и отправьте его в GitHub.
3. Обновите опубликованную ветку:

```bash
git push origin main:gh-pages
```

GitHub Pages публикует сайт из ветки `gh-pages`.

После публикации GitHub покажет ссылку на сайт в этом же разделе `Pages`.

## Локальный просмотр

Откройте `index.html` в браузере. Сборка и установка зависимостей не нужны.
